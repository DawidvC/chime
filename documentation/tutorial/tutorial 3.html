Built-In Classes

Chime features a few classes built into the runtime.  Here are a few that can be created with literal forms:

// Ranges
"hello, world"[0:6].print()
"hello, world"[7:-1].print()

// Arrays
["hello", ",", "world"].each() do (string)
{
    string.print()
}

// Hashes
hash = {"a":"world", "b":"hello", "c":","}
hash["b"].print()
hash["c"].print()
hash["a"].print()

More Method Types

There are actually four different method types you can define with Chime.  These different types have slightly different rules for execution and allow for different syntax.

Properties allow you to have convenient get and set access, supporting assignment with the equal operator.

property greeting()
{
    get
    {
        return "hello"
    }
}

"{self.greeting}, world".print()

Indexers give you array-style access and also include the ability to assign with equals.

indexer self[x]
{
    get
    {
        return "hello, {x}"
    }
}

self["everyone"].print()
