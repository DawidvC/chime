Multi-File Programs

It can be pretty hard to fit all of your code in one file. Chime's solution to this problem looks a lot like a scripting language. You simply write another file and import it. Chime automatically pre-compiles the code it needs and links things appropriately.

import "Tutorial"

greeter = Greeter.new()
greeter.greet()
