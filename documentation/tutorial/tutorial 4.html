More Method Types and Attributes

Classes can also define instance variables, and can include a few other types of methods.

implementation Greeter
{
    attribute addresse
    
    method greet()
    {
        "hello, {self.addresse}".print()
    }
    
    property addresse(value)
    {
        get
        {
            return addresse
        }
        set
        {
            addresse = value
        }
    }
}

greeter = Greeter.new()
greeter.addresse = "universe"

greeter.greet()

You can use the attribute keyword to define instance variables. A property is just like a method, but makes it possible to do assignments with the = operator.

Multi-File Programs

It can be pretty hard to fit all of your code in one file. Chime's solution to this problem looks a lot like a scripting language. You simply write another file and import it. Chime automatically pre-compiles the code it needs and links things appropriately.

import "Tutorial"

greeter = Greeter.new()
greeter.greet()
